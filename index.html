Planes

Tres formas claras de resolver el sitio web: una opción de presentación profesional, una opción administrada destacada y una opción de control total para quienes prefieren manejar su propio proveedor.

Plan Básico

$650 Pago único, sin renovaciones anuales

Ideal para negocios que quieren un sitio profesional de presentación, visual y bien resuelto para mostrar su marca, servicios, menú o catálogo.

Incluye:

  • Sitio web informativo profesional.
  • Dirección web estándar incluida.
  • Información, servicios, menú o catálogo.
  • Galería de imágenes.
  • Ubicación y contacto por WhatsApp.
  • Sin pagos anuales de dominio o alojamiento.

Este plan está pensado para sitios de presentación e información con muy buen nivel visual. Puede incluir secciones, galerías, menús, catálogos y contenido de marca, pero no procesos que requieran base de datos, panel administrativo o gestión de clientes. Si el cliente decide continuar con otro proveedor o con otro desarrollador, puede solicitar la entrega de los archivos y el contenido de su sitio. Los cambios o ajustes solicitados después de la entrega final se cotizan por separado.

Plan Plus Administrado

$1,300 El primer año

Un sitio profesional con la parte técnica resuelta dentro del servicio, ideal para quien quiere comodidad sin ocuparse del proveedor.

El precio del primer año incluye:

  • Creación del sitio web: $650.
  • Hosting administrado durante 12 meses: $450.
  • Configuración y uso de dirección web profesional: $200.

En este plan, el cliente invierte $1,300 MXN durante el primer año. Ese total ya incluye la creación del sitio web, la configuración inicial, el hosting administrado y la dirección web profesional.

La diferencia frente al Plan Control Total no está en la calidad del sitio web, sino en que aquí la administración técnica, el hosting y la dirección profesional quedan resueltos dentro del servicio.

Beneficios:

  • Dirección como umyr.sitio-web-oficial.com.
  • Certificado de seguridad.
  • Alojamiento y administración técnica incluidos.
  • Infraestructura compatible con base de datos.
  • Posibilidad de agregar funciones dinámicas.
  • Entrega de archivos y datos si el cliente solicita migración.
  • No es necesario administrar cuentas técnicas o servicios externos.

Después del primer año, la renovación es de $750 MXN al año para mantener activo el hosting administrado y la dirección web profesional.

En caso de cancelar el servicio, el cliente puede solicitar la entrega de los archivos y datos de su sitio para migrarlo con otro proveedor o con otro desarrollador. La dirección web en subdominio permanece como parte del servicio contratado porque pertenece a este esquema administrado. Los cambios o ajustes solicitados después de la entrega final se cotizan por separado.

Complemento opcional: Panel de gestión de contenido — +$550

Herramienta opcional de pago único. Se puede agregar si el cliente la desea en los planes Plus y Control Total.

Es una herramienta opcional para gestionar la información principal del sitio de forma directa, con la libertad de mantener la página actualizada sin depender del desarrollador en cada ajuste cotidiano.

Con este panel se puede:

  • Editar textos e información general del negocio.
  • Actualizar horarios, ubicación, teléfonos y datos de contacto.
  • Subir, reemplazar u organizar imágenes del sitio.
  • Modificar productos, servicios, platillos, descripciones y precios.
  • Mantener vigente el contenido comercial publicado.

Este panel está pensado para administrar el contenido ya existente del sitio. No incluye rediseño visual, nuevas secciones, inventario, sistemas de usuarios, pedidos avanzados ni funciones nuevas. Cualquier ampliación o desarrollo adicional se cotiza por separado.
